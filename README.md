# project-nomad-accessories

Scripts and documentation to help you get more out of [Project N.O.M.A.D.](https://github.com/Crosstalk-Solutions/project-nomad) — an offline AI assistant and self-hosted knowledge base designed for resilience and off-grid use.

## What's Here

**Scripts** — drop-in utilities that extend NOMAD's functionality:

- [`scripts/kibuild/`](scripts/kibuild/) — rebuilds the Kiwix library index after adding ZIM files, with optional custom icon support

**How-Tos** — guides for common tasks and workflows:

- [`howtos/Archiving-Large-YouTube-Channels/`](howtos/Archiving-Large-YouTube-Channels/) — download and package YouTube channels as offline ZIM files for Kiwix

## Contributing

If you've built something useful on top of NOMAD, feel free to open a PR.
